# JavaScript Basics Demo

This project demonstrates the basic concepts of JavaScript that I learned while starting my JavaScript journey.

## Topics Covered

- Variables (`let` and `const`)
- Data Types
- String Methods
- Template Literals
- Arrays
- Array Methods
- Objects
- Nested Objects
- Object Destructuring
- Console Output

## Technologies Used

- HTML5
- JavaScript (ES6)

## How to Run

1. Download or clone the project.
2. Open `index.html` in any web browser.
3. Open the browser's Developer Tools.
4. Go to the **Console** tab to view the output.

## Learning Purpose

This project was created to practice and understand JavaScript fundamentals before moving on to functions, loops, DOM manipulation, and events.
